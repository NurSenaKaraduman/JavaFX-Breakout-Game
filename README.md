## ⚡ Planlanan Mekanikler ve Evrim
| Özellik | Açıklama | Durum |
| :--- | :--- | :--- |
| **Durum Yönetimi** | Ana Menü ve Oyun Bitti (Game Over) ekranları | yükleniyor... |

## 🛠️ Teknik Detaylar
- **JavaFX Canvas:** Yüksek performanslı 2D render işlemleri için.
- **AnimationTimer:** 60 FPS sabit oyun döngüsü (Game Loop) yönetimi.
- **Nesne Yönelimli Tasarım:** Tuğlalar, top ve raket için modüler sınıflar.
